MANTENIMIENTO Y FALLAS DE SOFTWARE

1. PROGRAMA NO RESPONDE
- Forzar cierre desde Administrador de tareas.
- Reiniciar equipo.
- Reinstalar programa.
- Ejecutar como administrador.

2. SOFTWARE NO ABRE
Puede ser incompatibilidad, archivos corruptos o falta de permisos.

3. POSIBLE VIRUS
Síntomas:
- ventanas emergentes
- lentitud
- procesos desconocidos

Pasos:
- desconectar internet
- ejecutar antivirus
- eliminar temporales

4. EXCESO DE ARCHIVOS TEMPORALES
Produce lentitud general.

Pasos:
- usar %temp%
- eliminar temporales
- limpiar caché navegador
