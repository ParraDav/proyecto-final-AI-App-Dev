DIAGNÓSTICO DE HARDWARE BÁSICO

1. COMPUTADOR SE SOBRECALIENTA
Puede deberse a ventilación obstruida, ventiladores dañados o acumulación de polvo.

Pasos:
- Apagar el equipo.
- Limpiar ventiladores.
- Verificar pasta térmica.
- Usar base refrigerante.

2. BATERÍA NO CARGA
- Revisar cargador.
- Probar otro enchufe.
- Revisar conector.
- Evaluar desgaste de batería.

3. DISCO DURO CON RUIDOS
Puede indicar daño físico inminente.

Pasos:
- Hacer copia de seguridad.
- Ejecutar diagnóstico SMART.
- Reemplazar unidad.

4. MEMORIA RAM DEFECTUOSA
Síntomas:
- reinicios
- bloqueos
- pantalla azul

Pasos:
- limpiar módulos
- probar uno por uno
